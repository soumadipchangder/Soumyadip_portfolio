# Soumyadip Changder - AI/ML Engineer Portfolio

Welcome to the source code for my personal portfolio! This project showcases my experience, projects, skills, and certifications in the domains of Artificial Intelligence, Machine Learning, and Software Engineering.

## 🚀 Live Demo

[www.soumyadipchangder.com](https://soumyadipchangderportfolio.vercel.app/)

## 🛠️ Tech Stack

This project is built using:

- React.js
- JavaScript
- CSS (styled-components)

## 💻 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/soumadipchangder/masterPortfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd masterPortfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
